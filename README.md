# Helpdesk Interno

Aplicação fullstack para gestão de chamados internos da empresa.

## 📌 Tecnologias
- React + Vite
- TypeScript
- Tailwind CSS
- Node.js (futuro backend)

## 🚀 Funcionalidades
- Cadastro de chamados
- Acompanhamento de status
- Painel de controle
